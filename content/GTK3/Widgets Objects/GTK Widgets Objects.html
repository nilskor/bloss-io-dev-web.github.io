GTK Widgets and Objects

GTK is a widget toolkit library for creating graphical user interfaces (GUI). Each user interface (UI) is built by adding buttons, text labels, input fields, drop down menus, and other 'widgets' or 'objects' to a window. Those widgets and objects are described in further detail in this section.

Object Hierarchy

Widget toolkits and other object-based structures often contain a tree-like structure that describes the hierarchy of all objects. This is that tree structure, beginning with the most common ancestor for the GTK widgets - the 'GObject' root object.

Widget Gallery

Akin to a photo gallery, this is a quick summary of the visual representation of many of the most common widgets for the UI. It should be noted that many objects can be significantly themed via CSS to achieve any desired look for an application. Without any bespoke styling, the application will simply take on the look and feel of all the other applications within bloss OS.

GTKApplication Class

GtkApplication is a class that handles many important aspects of a GTK application in a convenient fashion, without enforcing a one-size-fits-all application model. While the class works fine with plain GtkWindows, it is recommended to use it together with GtkApplicationWindow.
