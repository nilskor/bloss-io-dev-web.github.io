Overview

GTK is a library for creating graphical user interfaces. Each user interface (UI) is built by adding buttons, text labels, input fields, drop down menus, and other 'widgets' to a window. It is also a cross-platform development tool that is designed to support many languages via bindings, such as Python, Javascript, C++, Rust, Perl, D and many more. It works on many UNIX-like platforms, Windows, and macOS.

Widgets and Objects

User Interfaces (UI) are built up using buttons, sliders, text-boxes and many other different types of widgets and objects to create the functionality required for immersive and engaging applications. It is in this section we explore the many different types available in the GTK library.
